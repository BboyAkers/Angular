To make this application work

1.)Go to https://github.com/settings/developers

2.)Click Register a new application

3.)Create application name
  3a.)Set homepage URL and Authorization callback URL to "http://localhost:3000/"

4.) Click register application

5.)Copy Client ID and navigate to /app/services/github.service.ts;

6.) Paste Client ID inside the quotations in private client_id: ''<---

7.) Repeat the 5 and 6 but do it for Client Secret

8.) npm start and your application should run!

ENJOY!! :)
